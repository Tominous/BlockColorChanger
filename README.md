# BlockColorChanger
minecraft spigot plugin

Right-click on the dye to change the color of the colored block.

## Corresponding block

- Wool
- Carpet
- Stained Glass
- Stained Glass Pane (Warning, there's a bug)
- Concrete Powder
- Terracotta (colored)
- Glazed Terracotta

## -----------------
Native Minecraft Version: 1.16
Tested Minecraft Versions:1.16
